# Salary-Predictor
This is a simple Machine Learning desktop application built using Python. It predicts salary based on years of experience using a Linear Regression model and provides a graphical user interface (GUI) using Tkinter.

## Features
 Predict salary based on experience
 <br>
 Simple and clean desktop GUI
 <br>
 Built using Linear Regression model
 <br>
 Real-time predictions
 <br>
 Beginner-friendly ML project

## Technologies Used
 Python
 <br>
 Tkinter (GUI)
 <br>
 Pandas
 <br>
 NumPy
 <br>
 Scikit-learn (Linear Regression)
